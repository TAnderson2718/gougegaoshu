# 腾讯云安全组配置

## 需要添加的入站规则

在腾讯云控制台 → 云服务器 → 安全组 → 入站规则中添加：

| 协议 | 端口 | 来源 | 策略 | 备注 |
|------|------|------|------|------|
| TCP | 3000 | 0.0.0.0/0 | 允许 | 前端React应用 |
| TCP | 3001 | 0.0.0.0/0 | 允许 | 后端API服务 |

## 当前已有规则（保持不变）
- HTTP (80) - Web服务
- HTTPS (443) - 安全Web服务  
- Linux登录 (22) - SSH访问
- Windows登录 (3389) - 远程桌面（如果需要）
- Ping - 网络连通性测试

## MySQL端口说明
- MySQL (3306) - 不要对外开放，仅内部访问
- 数据库应该只允许本地连接，不对公网开放

## 配置步骤
1. 点击"添加规则"
2. 选择"自定义"
3. 填入端口号（3000 或 3001）
4. 来源选择 0.0.0.0/0
5. 策略选择"允许"
6. 点击"完成"
